# Trendy

Does trend analysis on the output of the LA Code Analysis tool (the statistics file)

# Setup

Use Python 3.9.0

Install requirements.txt with `pip install -r requirements.txt`

# Run

```sh
./trendy.py <path to some stats file>
```

# Test

Use pytest to run the unit tests

Use behave to run the bdd tests

# Rules

Keep it light, keep it fast, keep it simple
